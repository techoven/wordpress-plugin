# WordPress Plugin: Brand My Login

This repository hosts the **Brand My Login** WordPress plugin developed by Techoven Solutions.

- Plugin code lives in `brand-my-login/`
- WordPress-friendly documentation is inside `brand-my-login/readme.txt`
- To install, copy the entire `brand-my-login` folder into `wp-content/plugins/` or zip that folder and upload it through the WordPress dashboard

For feature details, screenshots, and changelog, please refer to the plugin readme.
