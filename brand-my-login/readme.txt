A lightweight plugin by **Techoven Solutions** that replaces the default WordPress login branding with your own logo and colors.

== Description ==

Brand My Login keeps things simple: upload one logo, pick one brand color, and your `/wp-login.php` screen instantly looks like your company instead of WordPress. There are no upsells, no bloat, and it only loads on the login page for zero performance impact.

**Key features**

* Custom logo using the WordPress media library
* Brand color applied to the login button and footer links
* Clean settings page under **Settings » Brand My Login**
* Responsive logo scaling for desktop, tablet, and mobile
* Removing or deactivating the plugin reverts the login page automatically

== Installation ==

1. Upload the `brand-my-login` folder to `/wp-content/plugins/` or install via zip upload.
2. Activate **Brand My Login** through the **Plugins** menu.
3. Go to **Settings » Brand My Login**, upload your logo, pick a brand color, and hit **Save Changes**.

== Frequently Asked Questions ==

= What happens if I don’t select a logo? =
Nothing breaks—WordPress shows its default logo until you upload one.

= Does this plugin slow down my site? =
No. All CSS is injected only on the login screen, so front-end performance is untouched.

= Can I undo everything easily? =
Yes. Deactivating or deleting the plugin restores the original WordPress login page instantly.
